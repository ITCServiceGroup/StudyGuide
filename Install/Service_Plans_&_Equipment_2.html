GFiber Service Plans & Equipment Configurations

A detailed reference for authorized Customer Premise Equipment (CPE), firmware, and key installation considerations for each GFiber service plan.

Universal Installation Guidelines

Rules for All Modern Installations

  • The Wi-Fi 6E Router (GR6EXX0C) is the new standard for all new connects on Lifestyle and modern Legacy plans (1 Gig and higher).
  • For multi-gig plans (5 Gig and 8 Gig), always use CAT6A Ethernet cable for hardwired connections to achieve full speeds. For 2 Gig and lower, CAT5e is acceptable.
  • Wired backhaul is always preferred over wireless mesh for the best performance. If the customer allows it, run Ethernet for mesh extenders.
  • Technicians must ensure customers only have a single, unified SSID during install. Do not split bands (2.4/5/6 GHz) unless a customer has a specific, advanced need, as this leads to a poor user experience.
  • Always confirm the required ONT is installed. If it's incorrect, a Pro Install is required to swap it.

Installation Rules & Best Practices

  • The Ethernet cable from the Fiber Jack must be connected to the silver 10G WAN port.
  • Incompatible with the older pluggable ONT (GFLT210) used in some legacy 2 Gig setups.
  • For customers bringing their own router, the GR6EXX0C must still be installed first, then the customer can plug their router into it.

Installation Rules & Best Practices

  • Only install if needed based on home size (>2,000 sq. ft.) or customer's tested need. Do not install just because the customer asks for it.
  • Unneeded extenders can degrade Wi-Fi performance.
  • Place the extender roughly halfway between the router and the area with poor coverage.
  • Customers can purchase additional extenders for $85 each (max 6 per household).
